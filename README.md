# BTAudio

**English** | [简体中文](README.zh_CN.md)

Bluetooth audio playback (A2DP Sink) connector for Windows 10 2004+.

This project is a fork of [AudioPlaybackConnector](https://github.com/ysc3839/AudioPlaybackConnector) by [ysc3839](https://github.com/ysc3839), which stopped maintenance in 2020. Based on the original work, this version has been optimized and enhanced for continued use.

# Usage

* Download and run BTAudio from [releases](https://github.com/ysc3839/AudioPlaybackConnector/releases).
* Add a bluetooth device in system bluetooth settings. You can right click BTAudio icon in notification area and select "Bluetooth Settings".
* Click BTAudio icon and select the device you want to connect.
* Enjoy!
